# Open Data

**Open Data** sind Verwaltungsdaten, die maschinenlesbar, frei lizenziert und kostenlos für jeden zugänglich sind, egal ob Bürger:in, Journalist, Forschende oder Software-Entwicklerin.

## Open-Data-Prinzipien

| Prinzip | Bedeutung |
|---|---|
| **Frei zugänglich** | Ohne Anmeldung, ohne Bezahlung |
| **Maschinenlesbar** | JSON, CSV, XML. nicht nur PDF-Scans |
| **Strukturiert** | Saubere Spalten + Datentypen |
| **Offen lizenziert** | CC0, CC-BY, oder Datenlizenz Deutschland (dl-de-by-2.0) |
| **Aktuell** | Regelmäßiger Daten-Refresh, nicht ein 5-Jahres-Snapshot |

## Open-Data-Gesetz Bund + Länder

- **Bund:** Open-Data-Gesetz (OD-G), seit 2017. Pflicht zur Bereitstellung für Bundesbehörden.
- **Länder:** verschiedene Landes-Gesetze (Berlin, Hamburg, Bremen, Nordrhein-Westfalen, Baden-Württemberg haben eigene Open-Data-Gesetze).
- **Kommunen:** keine bundesweite Pflicht. Freiwillige Veröffentlichung.

## Plattformen

| Plattform | Wer dahinter |
|---|---|
| **GovData.de** | Bund + Länder + Kommunen, zentrale Aggregation |
| **daten.berlin.de** | Berlin (mit eigenen API) |
| **transparenz.hamburg.de** | Hamburg (besonders breit) |
| **opendata.muenchen.de** | München |
| **opendata.swk.de** | Stadtwerke Köln |

Jede Stadt mit eigenem Portal landet idealerweise auch auf GovData (Bundes-Aggregation).

## Häufige Datensätze

- **Bevölkerung**. Einwohnerstatistik, Altersstruktur
- **Verkehr**. Fahrpläne, Echtzeit-Daten, Baustellen
- **Bildung**. Schulstandorte, Schultests
- **Infrastruktur**. WLAN-Spots, EV-Ladestationen, Toiletten
- **Wahlen**. Wahlergebnisse pro Wahlbezirk
- **Bauleitplanung**. B-Pläne, Flächennutzungspläne
- **Umweltdaten**. Luftqualität, Lärm, Pegelstände

## In AmtsScore-Methodik (Dimension 5)

AmtsScore prüft pro Stadt:

- **Existenz eines Open-Data-Portals**: eigene Domain oder GovData-Integration
- **Anzahl der Datensätze**. Quantität als Indikator für Maturität
- **Aktualität des letzten Datensatzes**: wird das Portal noch gepflegt?

Eine Stadt ohne Open Data oder mit Portal-Leiche (letzter Eintrag 2020) bekommt einen niedrigen D5-Score, auch wenn andere Dimensionen gut sind.
